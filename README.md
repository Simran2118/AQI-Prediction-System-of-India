# AQI Prediction System

## Overview
This project predicts Air Quality Index (AQI) values using Machine Learning techniques.

## Features
- AQI Prediction
- Data Visualization
- User-Friendly Web Interface
- Machine Learning Based Analysis

## Technologies Used
- Python
- Flask
- Pandas
- NumPy
- Scikit-Learn
- HTML
- CSS
- JavaScript

## Dataset
Air quality dataset containing PM2.5, SO2, NO2, CO, O3 and location information.

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python app.py

3. Open:
   http://127.0.0.1:5000
